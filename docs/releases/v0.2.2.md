# NewGodWar v0.2.2

NewGodWar v0.2.2는 일부 능력의 타깃 지정 방식과 범위 판정을 조정한 능력 사용성 개선 릴리즈입니다.

## Highlights

- `counter` 일반 스킬이 `/x <플레이어>`로 지정한 적을 거리 제한 없이 대상으로 삼도록 변경했습니다.
- `hecate` 우클릭 스킬이 지정 타깃 기반으로 동작하도록 조정했습니다.
- `morpious` 좌클릭 스킬이 시야 판정 대신 지정 타깃을 사용하며, 타깃 지정 필요 능력으로 표시됩니다.
- `yugwansun` 강화 스킬의 적 대상 판정을 전체 적 플레이어 기준으로 확장했습니다.
- 공용 능력 기반 클래스에 전체 적 플레이어 조회와 지정 타깃 접근 헬퍼를 추가했습니다.

## 이전 버전 대비 주요 변경점

### 타깃 지정 개선

- `counter`, `hecate`, `morpious`가 시야 또는 짧은 거리 판정에 막히지 않고 `/x <플레이어>`로 지정한 타깃을 더 일관되게 사용합니다.
- 대상이 없거나 같은 팀인 경우에는 기존 실패 메시지 흐름을 유지합니다.
- `morpious`는 능력 정보에서 타깃 지정 명령을 사용하는 능력으로 처리됩니다.

### 능력 범위 조정

- `yugwansun` 강화 스킬은 주변 14칸 적이 아니라 현재 접속 중인 전체 적 플레이어에게 디버프를 적용합니다.
- 아군 버프 범위는 기존처럼 주변 14칸과 본인 기준을 유지합니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.2.2.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
