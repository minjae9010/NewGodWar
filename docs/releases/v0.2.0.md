# NewGodWar v0.2.0

NewGodWar v0.2.0은 장영실 능력의 역할을 재정리하고, Themachy 호환 능력 명령의 일반 사용자 흐름을 다듬은 마이너 릴리즈입니다.

## Highlights

- 장영실 일반 스킬을 부품 누적형 철 곡괭이 제작 능력으로 변경했습니다.
- 장영실 패시브가 철괴 공격 대신 곡괭이 공격에서 추가 피해와 감속을 부여하도록 바뀌었습니다.
- `/t a` 계열 능력 명령에서 일반 사용자도 능력 확인/도움말 흐름을 더 자연스럽게 사용할 수 있게 했습니다.
- 능력 명령 자동완성이 관리자 권한 여부에 따라 플레이어 이름과 능력 ID 제안을 구분합니다.

## 이전 버전 대비 주요 변경점

### 장영실

- 일반 스킬 비용을 조약돌 12개, 쿨타임 60초로 조정했습니다.
- 일반 스킬을 사용할 때 철 곡괭이 부품을 1개씩 제작합니다.
- 부품 3개가 모이면 철 곡괭이 1개를 지급합니다.
- 패시브는 곡괭이로 공격할 때 낮은 확률로 추가 피해와 감속을 부여합니다.
- 설명 문구를 채굴/진입 보조 역할에 맞춰 정리했습니다.

### 명령어

- 능력 그룹 명령에서 인자 부족 시 일반 능력 확인 흐름으로 자연스럽게 연결합니다.
- `/t a help`가 관리자 전용으로 막히지 않도록 정리했습니다.
- `/t a <player>` 형태의 일반 사용자 능력 확인과 관리자용 수동 지정 흐름을 더 명확히 분리했습니다.
- 자동완성에서 비관리자는 확인 가능한 대상 중심으로, 관리자는 능력 ID까지 제안받습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.2.0.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
