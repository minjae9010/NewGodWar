# NewGodWar v0.2.1

NewGodWar v0.2.1은 신규 능력 2종과 게임 시작 기본 지급 아이템 관리 기능을 추가한 운영 편의 릴리즈입니다.

## Highlights

- `nature` 자연계 능력을 추가했습니다.
- `siksin` 식신 능력을 추가했습니다.
- 게임 시작 기본 지급 아이템을 `game.skyblock-items` 설정으로 관리할 수 있게 했습니다.
- `/godwar defaultitems` 명령과 기본 지급 아이템 창고 GUI를 추가했습니다.
- 설정 GUI에서 스카이블럭 아이템 지급 항목을 우클릭하면 기본 지급 아이템 창고를 열 수 있습니다.
- 능력 명령과 시작 아이템 관련 문서를 갱신했습니다.

## 이전 버전 대비 주요 변경점

### 신규 능력

- `nature`: 무작위 식물을 얻고, 팀원에게 치유 버프를 부여하며, 주변 식물 상태에 따라 버프/디버프를 받습니다.
- `siksin`: 신성한 음식을 만들어 자신과 팀원을 강화합니다. 팀원이 음식을 먹으면 식신도 같은 버프를 얻습니다.

### 기본 지급 아이템

- `game.skyblock-items` 설정을 추가했습니다.
- 기본값은 `COOKED_BEEF` 64개입니다.
- `/godwar defaultitems`로 가상 창고를 열어 게임 시작 시 지급할 아이템을 편집할 수 있습니다.
- 콘솔/명령 기반으로 `list`, `add`, `set`, `remove`, `clear`, `reset`을 사용할 수 있습니다.
- 기존 `game.give-skyblock-items`가 켜져 있을 때만 기본 지급 아이템이 지급됩니다.

### 문서와 UI

- README, Wiki Home, 명령어, 설정 문서에 기본 지급 아이템 관리 기능을 추가했습니다.
- 설정 GUI의 스카이블럭 아이템 지급 항목에 기본 지급 아이템 창고 진입 동작을 추가했습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.2.1.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
