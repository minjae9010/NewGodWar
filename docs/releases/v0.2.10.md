# NewGodWar v0.2.10

NewGodWar v0.2.10은 능력 전투 판정과 생명주기를 전반적으로 안정화하고 Minecraft 26.2 호환성을 추가한 패치 릴리즈입니다.

## Highlights

- 폭발, 번개, 즉사 주문, 나락 이동도 킬타임과 아군 공격 방지 규칙을 따릅니다.
- 능력으로 발생한 환경 피해와 스탠스 전환 피해도 올바른 사용자에게 처치 기록이 귀속됩니다.
- 재접속, 관전 해제, 능력 봉인 해제로 시작 장비를 반복 획득할 수 없게 했습니다.
- 스크루지 능력이 사라진 뒤에도 팀 비용 할인이 영구적으로 남던 문제를 수정했습니다.
- 투사체 피격에도 반사, 회피, 공격자 약화 같은 방어 패시브가 정상 발동합니다.
- Minecraft 26.2 Paper 서버를 공식 호환성 및 릴리즈 검증 대상에 추가했습니다.

## 이전 버전 대비 주요 변경점

### 능력 전투 판정

- 능력 폭발과 번개에 사용자를 연결해 킬타임, 친선 공격, 참가자 상태를 동일하게 검사합니다.
- 환경 피해 또는 나락 이동 후 사망해도 킬 수와 니케 같은 처치 패시브가 정상 처리됩니다.
- 즉사 주문은 직접 체력 변경 대신 출처가 있는 치명 피해를 사용합니다.
- 고정 피해를 대상의 현재 체력으로 제한하던 처리를 제거해 방어구 착용 시 피해가 비정상적으로 줄어들지 않습니다.
- 공격 대상에서 비참가자, 관전자, 탈락자, 다른 월드 플레이어를 제외합니다.
- 아프로디테, 아폴론, 마법사, 부두술사 등 별도 대상 탐색을 사용하던 능력도 공통 보호 규칙을 적용합니다.

### 투사체와 피해 패시브

- 투사체 적중 시 공격자의 전용 투사체 효과와 피해자의 피격 패시브를 모두 처리합니다.
- 반사 능력끼리 피해가 무제한으로 재진입하지 않도록 반사 중첩을 차단합니다.
- 스탠스가 원래 공격자를 잃지 않으면서 피해 증폭과 넉백을 제거하도록 처리 흐름을 수정했습니다.

### 능력 생명주기와 장비

- 지속 효과 복구용 `onAssign`과 시작 장비 지급용 `onPrepare` 생명주기를 분리했습니다.
- 안중근, 정원사, 낚시꾼, 해리, 헤르미온느, 세종, 스나이퍼, 사이코스노우, 토르, 이순신의 시작 장비는 게임 준비 시에만 지급됩니다.
- 진행 중 능력 변경과 중간 참여에서는 시작 장비를 지급하면서 재접속과 봉인 해제에서는 장비를 재지급하지 않습니다.
- 외부 능력 개발자를 위한 `onPrepare` 사용법을 개발 문서에 추가했습니다.

### 능력별 안정화

- 스크루지 할인은 정적 상태를 저장하지 않고 현재 팀의 활성 스크루지 능력을 기준으로 계산합니다.
- 잭프로스트 얼음 구체가 겹치거나 블록이 변경됐을 때 얼음이 영구적으로 남거나 새 블록을 덮어쓰지 않도록 복원 조건을 보강했습니다.
- 일반 능력 쿨타임을 잘못된 슬롯에 기록하던 능력들을 수정해 사이드바와 쿨타임 안내가 일치합니다.
- 아프로디테는 발동 조건과 유효 대상을 먼저 검사한 뒤 비용과 쿨타임을 사용합니다.

## 지원 버전

- Paper 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 MC 26.2까지를 지원 대상으로 합니다.
- 서버 버전에 맞는 Java 8, 16, 17, 21 또는 25가 필요합니다.
- Spigot/CraftBukkit에서도 플러그인은 로드되지만 자동 게임룰 적용 및 복구는 안전을 위해 건너뜁니다.
- MC 1.11.x 이하 Legacy 서버와 Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.2.10.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean test build`
- 킬타임 모드, 서버 감지, 능력 피해 출처 유효기간 단위 테스트 8개
- Paper 26.2 / Java 25 시작 스모크 테스트
- 전체 지원 Paper 버전 매트릭스는 별도 호환성 워크플로우에서 계속 확인합니다.
