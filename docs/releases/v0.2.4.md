# NewGodWar v0.2.4

NewGodWar v0.2.4는 도박 보상 설정을 단일 보상 목록으로 정리하고, 오래된 타짜 전용 보상 설정을 자동 정리하는 운영 편의 릴리즈입니다.

## Highlights

- 도박 GUI가 능력별 보상 분기 없이 `gambling.rewards.normal` 보상 목록만 사용하도록 단순화되었습니다.
- `gambling.rewards.tajja` 기본 설정을 제거했습니다.
- 기존 서버 설정에 남아 있는 `gambling.rewards.tajja` 값은 플러그인 시작 또는 `/godwar reload` 시 자동 제거됩니다.
- 설정 GUI와 `/godwar gamblereward` 명령 도움말에서 타짜 전용 보상 편집 항목을 제거했습니다.
- `plugin.yml` 리소스 확장에 문자열 버전을 사용하도록 조정했습니다.

## 이전 버전 대비 주요 변경점

### 도박 보상 설정

- 도박 보상은 이제 일반/타짜로 나뉘지 않고 하나의 보상 목록으로 관리됩니다.
- `/godwar gamblereward <normal> <번호|add> hand|message|<material> [값]` 형식으로 보상 아이템과 메시지를 편집합니다.
- 설정 GUI의 도박 상품 화면도 단일 상품 편집 화면으로 정리되었습니다.

### 레거시 설정 정리

- `config.yml`에 오래된 `gambling.rewards.tajja` 항목이 있으면 자동으로 제거하고 저장합니다.
- `/godwar reload`와 설정 GUI 다시 불러오기에서도 같은 정리 로직을 실행합니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.2.4.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
