# NewGodWar v0.2.6

NewGodWar v0.2.6은 월드 관리 문서 링크를 정리하고, 관리 월드 로드 시 월드 폴더 감지 방식을 보강한 안정화 릴리즈입니다.

## Highlights

- Wiki 문서 링크에서 파일 확장자와 경로 표기를 정리했습니다.
- Home, Sidebar, Configuration, Index 문서의 내부 링크를 일관되게 맞췄습니다.
- 관리 월드 자동 로드와 `/godwar world load`가 `WorldBackupManager`의 월드 폴더 감지 로직을 함께 사용하도록 조정했습니다.
- 일반 월드 폴더뿐 아니라 서버/버전별 월드 폴더 배치 차이를 더 안정적으로 처리합니다.

## 이전 버전 대비 주요 변경점

### 문서 링크 정리

- Wiki 페이지 간 링크를 확장자 없는 경로 형식으로 정리했습니다.
- 월드 관리와 설정 문서의 상호 링크가 같은 규칙을 사용하도록 맞췄습니다.

### 월드 폴더 감지

- 서버 시작 시 `world.managed-worlds`에 등록된 월드를 자동 로드할 때 공용 월드 폴더 탐색 로직을 사용합니다.
- `/godwar world load <world>`도 같은 로직을 사용해, 단순한 서버 루트 폴더 검사만으로 놓칠 수 있는 월드 폴더를 더 잘 찾습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.2.6.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
