# NewGodWar v0.2.8

NewGodWar v0.2.8은 능력 확정 대기와 게임 월드 복원 흐름을 안정화하고, 코어 보호 해제 시간을 더 세밀하게 조정할 수 있게 한 기능 릴리즈입니다.

## Highlights

- 모든 플레이어가 능력을 확정하거나 재추첨 기회를 모두 사용하면 능력 선택 대기가 자동으로 종료됩니다.
- 오프라인, 관전자, 팀 이탈 플레이어는 능력 선택 대기 목록에서 제거되어 게임 시작이 막히지 않습니다.
- 게임 월드 자동 초기화가 시작 직전 스냅샷을 저장하고 종료 후 언로드, 복원, 재로드하는 흐름으로 보강되었습니다.
- `core.explosion-unlock-seconds` 설정을 추가해 게임 시작 후 일정 시간이 지나면 폭발로 코어를 파괴할 수 있게 했습니다.
- 나무, 돌, 철, 다이아 곡괭이의 코어 파괴 허용 기본 시간을 1500초로 설정하고, 금 곡괭이는 허용 대상에서 제외했습니다.
- 아이올로스와 아카샤의 아군 버프가 자기 자신에게도 적용됩니다.
- 사제 축복은 이제 최소 하나의 효과를 반드시 부여합니다.
- 내장 능력 수를 실제 목록에 맞춰 90개로 정리하고 미사용 능력 파일을 제거했습니다.
- 설정 GUI에 코어 폭파 허용 시간과 Locator Bar 게임룰 토글을 추가했습니다.

## 이전 버전 대비 주요 변경점

### 능력 선택과 게임 시작

- `/gw yes`로 능력을 확정한 플레이어가 모두 준비되면 확정 대기가 자동 종료되고 시작 카운트다운이 짧아집니다.
- `/gw no` 재추첨 기회를 모두 사용한 플레이어도 확정 처리 후 자동 종료 조건에 포함됩니다.
- 게임 준비 중 팀에서 빠지거나 관전자로 전환되거나 접속이 끊긴 플레이어는 대기 목록에서 정리됩니다.

### 게임 월드 복원

- 게임 월드 자동 초기화가 시작 직전 활성 게임 월드를 기준으로 스냅샷을 저장합니다.
- 게임 종료 시 해당 월드를 언로드한 뒤 스냅샷을 복원하고 다시 로드합니다.
- 로비 월드이거나 플레이어가 남아 있는 월드는 안전하게 초기화를 건너뜁니다.

### 코어 보호 설정

- `core.explosion-unlock-seconds` 기본값은 1800초이며, `-1`로 설정하면 폭파 보호가 자동 해제되지 않습니다.
- 폭파 보호 시간이 지나면 전체 안내와 타이틀로 폭발 코어 파괴 허용 상태를 알립니다.
- 곡괭이별 기본 허용 시간은 1500초이며, 금 곡괭이는 명령어와 GUI 허용 대상에서 제외됩니다.
- 월드 / 코어 설정 GUI의 코어 보호 설정에서 곡괭이 허용 시간과 폭파 허용 시간을 함께 조정할 수 있습니다.

### 능력과 문서

- 아이올로스와 아카샤의 일반 능력 설명과 실제 동작을 맞춰 자기 자신도 버프 대상에 포함했습니다.
- 아카샤의 고급 능력은 대상이 없을 때 비용을 쓰지 않고 안내 메시지를 보냅니다.
- 사제 축복은 무작위 결과가 비어도 재생 효과를 부여해 항상 하나 이상의 축복을 보장합니다.
- 능력 목록에서 제거된 미사용 능력을 정리하고 문서의 내장 능력 수를 90개로 수정했습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.
