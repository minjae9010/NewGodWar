# NewGodWar v0.1.4

NewGodWar v0.1.4는 능력 등급과 신규 능력, 서버 실행 중 업데이트 준비 기능, Wiki/이슈 템플릿 정리를 포함한 운영 편의 릴리즈입니다.

## Highlights

- 모든 내장 능력에 밸런스 기준 등급을 부여하고, 능력 도감/명령/스코어보드에서 등급을 확인할 수 있게 했습니다.
- 헤카테, 헤라, 니케, 페르세포네, 포세이돈 능력을 추가했습니다.
- `/godwar update [check|download]` 명령과 자동 업데이트 확인 기능을 추가했습니다.
- 팀 탈락 시 플레이어 처리 방식을 `spectator`, `kick`, `midjoin` 중 선택할 수 있게 했습니다.
- GitHub Wiki 문서, 이슈 템플릿, PR 템플릿을 정리했습니다.
- Paper 테스트 스크립트가 Mojang server jar를 미리 내려받아 오래된 Paperclip 실행 실패를 줄이도록 개선했습니다.

## 이전 버전 대비 주요 변경점

### 능력 시스템

- `AbilityGrade`를 추가하고 `@AbilityInfo`에서 능력 등급을 선언할 수 있게 했습니다.
- 능력 GUI, 능력 검색, 능력 정보 명령에 등급 표시와 검색 조건을 추가했습니다.
- 기본 내장 능력들의 등급을 S/A/B/C/D 기준으로 채웠습니다.
- 헤카테, 헤라, 니케, 페르세포네, 포세이돈 능력을 신규 등록했습니다.

### 게임 진행과 운영

- 팀 탈락 상태를 추적하고, 탈락 팀 플레이어를 관전, 퇴장, 중간 참여 중 설정값에 따라 처리합니다.
- `game.eliminated-player-action` 설정과 설정 GUI 항목을 추가했습니다.
- `/godwar info`와 점수판에서 탈락 팀 및 능력 등급 정보를 더 잘 확인할 수 있게 했습니다.
- `/godwar midjoin` 흐름을 탈락 팀 처리와 함께 정리했습니다.

### 업데이트

- GitHub Releases 최신 버전을 확인하는 `PluginUpdater`를 추가했습니다.
- 서버 시작 후 자동 확인, 관리자 접속 알림, 자동 다운로드를 `updates` 설정으로 제어할 수 있습니다.
- 새 릴리즈 jar는 서버 실행 중 `plugins/update/`에 준비되고, 서버 재시작 후 적용할 수 있습니다.
- `/godwar update check`, `/godwar update download` 명령을 추가했습니다.

### 문서와 릴리즈 품질

- GitHub Wiki용 시작하기, 게임 진행, 명령어, 설정, 능력 개발, 문제 해결 문서를 추가했습니다.
- Wiki 동기화 GitHub Actions 워크플로와 초기화 안내를 보강했습니다.
- 이슈 템플릿과 PR 템플릿을 추가해 버그, 호환성, 기능 요청 정보를 더 일관되게 받을 수 있게 했습니다.
- Paper 테스트 스크립트가 PaperMC Fill API와 Mojang version manifest를 활용하도록 개선했습니다.
- Java/Paper 호환성 매트릭스와 최신 지원 버전 검증 흐름을 갱신했습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.1.4.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
