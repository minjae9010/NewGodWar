# NewGodWar v0.1.6

NewGodWar v0.1.6은 킬타임 공격 금지 타이머와 업데이트 적용 흐름을 다듬고, 헤파이토스 용암 능력의 월드 정리를 보강한 운영 개선 릴리즈입니다.

## Highlights

- 게임 시작 후 공격 금지 시간을 `game.killtime-seconds`로 설정할 수 있게 했습니다.
- 킬타임 남은 시간을 스코어보드와 보스바에서 카운트다운으로 표시합니다.
- 킬타임 종료 시 공격 가능 안내 방송을 추가했습니다.
- 업데이트 jar 다운로드 후 설정에 따라 Bukkit reload로 바로 적용할 수 있게 했습니다.
- 헤파이토스 임시 용암이 만든 흑요석/조약돌/돌과 주변 물/공기 상태를 더 깔끔하게 복구합니다.
- 헤파이토스 일반 능력에 10초 쿨타임을 부여했습니다.

## 이전 버전 대비 주요 변경점

### 게임 진행

- `game.killtime-seconds` 설정을 추가했습니다. 기본값은 300초입니다.
- 게임 시작 후 킬타임 동안 스코어보드에는 남은 공격 금지 시간이 표시됩니다.
- `game.killtime-bossbar`가 켜져 있으면 보스바도 남은 시간 비율에 맞춰 줄어듭니다.
- 킬타임이 0초가 되면 보스바를 정리하고 "이제 공격할 수 있습니다" 안내를 방송합니다.
- 중간 참여 시 이미 배정된 능력이 있으면 기존 능력을 유지하고 재적용합니다.

### 업데이트 적용

- `updates.apply-without-restart` 설정을 추가했습니다. 기본값은 `true`입니다.
- `updates.apply-delay-seconds` 설정을 추가해 다운로드 완료 후 reload까지 대기 시간을 조절할 수 있습니다.
- 업데이트 다운로드가 끝나면 서버 전체 재시작 대신 Bukkit reload로 적용하는 흐름을 추가했습니다.
- `/godwar status`의 업데이트 상태 표시가 reload 대기/재시작 대기를 구분합니다.

### 능력 개선

- 헤파이토스의 임시 용암 생성 후 주변 3x3x3 영역에서 임시 부산물을 복구하도록 정리했습니다.
- 물과 용암이 만나 생긴 흑요석, 조약돌, 돌을 원래 공기/물 상태로 되돌리는 처리를 추가했습니다.
- 헤파이토스 일반 스킬 쿨타임을 문서와 구현 모두 10초로 맞췄습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.1.6.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
