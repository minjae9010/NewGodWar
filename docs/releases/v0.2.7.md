# NewGodWar v0.2.7

NewGodWar v0.2.7은 게임 맵 선택과 맵별 스폰/심장 설정을 추가하고, 진행 중 운영 명령과 능력 밸런스를 보강한 기능 릴리즈입니다.

## Highlights

- `/godwar map [world|clear]` 명령으로 이번 게임에 사용할 맵을 확인, 선택, 해제할 수 있습니다.
- 선택된 맵의 팀 스폰과 심장을 `maps.<world>` 아래에 별도로 저장합니다.
- 기존 `world.game-world`, `spawns`, `temples` 설정을 사용하는 서버는 선택된 맵과 위치 월드가 같을 때 맵별 설정으로 자동 마이그레이션됩니다.
- `/godwar changeteam <player> <team>` 또는 `/godwar changeteam <team> <player>` 명령으로 진행 중 플레이어의 능력과 인벤토리를 유지한 채 팀을 변경할 수 있습니다.
- `/godwar pickaxe [status]`와 `/godwar pickaxe <wooden|stone|iron|gold|diamond|all> <open|off|분>` 명령으로 곡괭이별 코어 파괴 허용 상태를 확인하고 조정할 수 있습니다.
- 여러 능력의 짧은 효과 지속 시간을 늘리고 강도를 조정해 전투 흐름을 다듬었습니다.
- `updates.auto-download` 기본값을 `false`로 바꿔 자동 다운로드 동작을 더 보수적으로 조정했습니다.

## 이전 버전 대비 주요 변경점

### 맵 선택과 마이그레이션

- `/godwar map`은 현재 선택된 맵과 선택 가능한 맵 목록을 보여줍니다.
- `/godwar map <world>`는 로드된 월드를 선택하며, 월드 폴더만 있는 경우 자동 로드 후 선택합니다.
- `/godwar map clear`는 선택된 맵을 해제합니다.
- 맵이 선택된 상태에서 `/godwar setspawn <team>`과 `/godwar settemple <team>`을 실행하면 해당 맵 전용 스폰/심장으로 저장됩니다.
- 기존 서버의 `spawns`와 `temples`가 선택된 게임 맵과 같은 월드를 가리키면 `maps.<world>` 설정으로 자동 복사됩니다.

### 운영 명령

- `/godwar changeteam`으로 진행 중 팀 변경을 처리할 수 있습니다. 변경 대상 플레이어의 능력과 인벤토리는 유지됩니다.
- `/godwar pickaxe status`는 게임 진행 시간과 곡괭이별 코어 파괴 허용 상태를 보여줍니다.
- `/godwar pickaxe <종류|all> open`은 즉시 허용, `off`는 자동 해제 없음, 숫자는 분 단위 해제 시간으로 저장합니다.

### 밸런스와 문서

- 여러 능력의 2초 내외 효과를 3초 중심으로 조정하고 일부 효과 강도를 정리했습니다.
- 게임 진행 가이드에 심장 파괴 규칙과 곡괭이 해제 조건을 보강했습니다.
- 명령어, 설정, 월드 관리 문서에 맵 선택과 새 운영 명령을 추가했습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.
