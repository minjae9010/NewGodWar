# NewGodWar v0.1.10

NewGodWar v0.1.10은 업데이트 적용 흐름을 재시작 기준으로 정리하고, 코어/곡괭이 진행 안내와 여러 능력의 밸런스 및 설명을 다듬은 운영 안정화 릴리즈입니다.

## Highlights

- 업데이트 jar 다운로드 후 Bukkit reload로 즉시 적용하던 흐름을 제거하고, 서버 재시작 후 적용 방식으로 정리했습니다.
- 곡괭이 허용 시간이 열릴 때 전체 안내 방송과 타이틀을 표시합니다.
- 코어 폭파 보호가 꺼져 있을 때 폭발로 파괴된 다이아 심장도 팀 탈락으로 처리합니다.
- 레거시 설정의 블레이즈 막대기 조합 팁을 자동 보정합니다.
- 헤파이토스, 프로스트, 텔레포터, 저격수, 해리포터/헤르미온느 등 여러 능력의 동작과 설명을 최신 밸런스에 맞게 정리했습니다.

## 이전 버전 대비 주요 변경점

### 운영과 업데이트

- `updates.apply-without-restart`는 더 이상 사용하지 않으며 기본값을 `false`로 변경했습니다.
- 업데이트 다운로드 완료 시 reload 대신 "서버를 재시작하면 적용" 안내를 방송합니다.
- `/godwar update [check|download]` 문서와 설정 문서를 재시작 적용 기준으로 갱신했습니다.
- 기존 config의 오래된 블레이즈 막대기 조합 팁 문구를 서버 시작 시 자동 보정합니다.

### 게임 진행

- 곡괭이 종류별 해제 시간이 도달하면 방송, 타이틀, 레벨업 사운드로 플레이어에게 알려줍니다.
- 폭발로 등록된 다이아 심장이 파괴될 때도 팀 탈락 판정을 수행합니다.
- 코어 폭파 보호 설정이 켜져 있으면 기존처럼 심장 블록을 보호합니다.

### 능력 밸런스

- 헤파이토스 용암 생성이 개별 능력 구현으로 정리되고 비용/쿨타임이 조정되었습니다.
- 프로스트 얼음 구체와 대상 속박 설명, 비용, 쿨타임을 구체화했습니다.
- 텔레포터는 안전 위치 이동과 시야 안 아군 교환 흐름을 명확히 했습니다.
- 저격수는 저격 준비 시간, 고속 화살, 리스폰 화살 보급 설명을 정리했습니다.
- 해리포터/헤르미온느 주문 설명과 보호 주문 흐름을 보강했습니다.
- 암살자, 대장장이, 아프로디테, 아이올로스 등 다수 능력의 비용/쿨타임/효과 설명을 최신 구현에 맞췄습니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.1.10.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

## 검증

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
