# NewGodWar v0.2.3

NewGodWar v0.2.3은 게임 시작 기본 지급 아이템을 실제 스카이블럭 시작 세트에 맞춘 설정 개선 릴리즈입니다.

## Highlights

- 기본 지급 아이템 초기값을 고기만 주는 구성에서 스카이블럭 시작 세트로 조정했습니다.
- 새 기본 설정은 용암 양동이, 얼음, 참나무 묘목, 뼛가루, 상자, 익힌 소고기를 함께 지급합니다.
- `/godwar defaultitems reset` 메시지와 자동완성 추천을 새 기본 세트에 맞췄습니다.
- 설정 문서와 명령어 문서의 기본 지급 아이템 예시를 갱신했습니다.

## 이전 버전 대비 주요 변경점

### 기본 지급 아이템

- 새 기본 설정과 `/godwar defaultitems reset`은 용암 양동이 2개, 얼음 2개, 참나무 묘목 1개, 뼛가루 4개, 상자 1개, 익힌 소고기 64개를 지급합니다.
- 참나무 묘목과 뼛가루에는 구버전 서버 호환을 위한 `legacy-material` 값이 함께 저장됩니다.
- 기존 서버의 `config.yml`에 이미 저장된 `game.skyblock-items` 값은 유지됩니다.
- 새 기본값을 쓰려면 `/godwar defaultitems reset`을 실행하면 됩니다.

### 운영 편의

- `/godwar defaultitems add`와 `/godwar defaultitems set` 자동완성이 새 기본 지급 아이템 후보를 우선 제안합니다.
- 초기화 완료 메시지가 특정 아이템 하나가 아니라 기본 스카이블럭 세트로 초기화했다는 내용으로 표시됩니다.

## 지원 버전

- Paper/Spigot 계열 MC 1.12.x부터 MC 1.21.x, 그리고 MC 26.1.1부터 26.1.2까지를 지원 대상으로 합니다.
- 최신 지원 Paper 버전에 대한 별도 스모크 테스트를 함께 실행합니다.
- MC 1.11.x 이하 Legacy 서버는 공식 지원 대상이 아닙니다.
- Pre-release / RC 서버는 공식 지원 대상이 아닙니다.

## 다운로드

아래 Assets에서 `NewGodWar-0.2.3.jar`를 다운로드한 뒤 서버의 `plugins` 폴더에 넣어 주세요.

- `./gradlew --no-daemon clean build`
- 최신 지원 Paper / Java 25 시작 스모크 테스트
- 레거시 Paper 버전 매트릭스는 별도 호환성 워크플로우로 계속 확인합니다.
