# NewGodWar Wiki

NewGodWar는 마인크래프트 신들의 전쟁 스타일 팀 기반 미니게임 플러그인입니다. 이 wiki는 서버 운영자가 게임을 설치하고, 팀/신전/능력을 설정하고, 실제 경기를 진행하는 데 필요한 기본 문서를 모아둔 공간입니다.

## 현재 기능 요약

- 다중 팀 자동/수동 배정과 팀 색상 스코어보드
- 팀별 스폰과 다이아몬드 심장 등록
- 심장 파괴 시 팀 탈락, 마지막 생존 팀 승리
- 랜덤 능력 배정, 재추첨, 수동 지정, 능력 도감 GUI
- 블레이즈 막대 기반 일반/고급 능력 사용
- 타깃형 능력 대상 지정, 팀 채팅, 관전 모드
- 도박 GUI와 일반/타짜 보상 설정
- 우르프 모드, 능력 블랙리스트, 게임룰 적용/복구
- Paper 버전별 스모크 테스트와 GitHub Actions 릴리즈 자동화

## 지원 범위

플러그인은 Java 8 바이트코드로 빌드되며, Paper/Spigot 계열 서버에서 동작하도록 설계되어 있습니다. 현재 호환성 스모크 테스트 대상은 MC 1.12.x부터 1.21.x, 그리고 26.1.1부터 26.1.2까지입니다.

| 서버 버전대 | 권장 Java |
| --- | --- |
| 1.12.x ~ 1.16.5 | Java 8 |
| 1.17.x | Java 16 |
| 1.18.x ~ 1.20.4 | Java 17 |
| 1.20.5 ~ 1.21.x | Java 21 |
| 26.1.x | Java 25 |

MC 1.11.x 이하 Legacy 서버와 Pre-release/RC 서버는 공식 지원 대상이 아닙니다.
