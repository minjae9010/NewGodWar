# 게임 진행 가이드

NewGodWar의 기본 승리 조건은 자기 팀의 다이아몬드 심장을 지키면서 다른 팀의 심장을 파괴하는 것입니다. 심장이 파괴된 팀은 탈락하고, 마지막으로 남은 팀이 승리합니다.

## 게임 상태

| 상태 | 의미 |
| --- | --- |
| 대기 | 팀, 스폰, 심장 등 시작 설정을 준비하는 상태 |
| 준비중 | `/godwar start` 이후 능력 배정과 재추첨 확정을 기다리는 상태 |
| 진행중 | 카운트다운이 끝나고 실제 전투가 진행되는 상태 |
| 종료 | 게임이 끝났거나 `/godwar stop`으로 중단된 상태 |

## 팀과 신전

기본 팀은 `red`, `blue`, `green`입니다. `config.yml`의 `teams`에서 팀 표시 이름, 색상, 활성 여부를 조정할 수 있고, 설정 GUI에서도 주요 팀 설정을 관리할 수 있습니다.

각 활성 팀에는 두 가지 위치가 필요합니다.

| 위치 | 설정 방법 | 설명 |
| --- | --- | --- |
| 팀 스폰 | `/godwar setspawn <team>` | 게임 시작과 리스폰 시 이동할 위치 |
| 다이아 심장 | `/godwar settemple <team>` | 바라보는 다이아몬드 블록을 팀 심장으로 등록 |

활성 팀의 스폰이나 심장이 빠져 있으면 게임 시작이 거부됩니다.

## 시작 흐름

1. 관리자가 팀과 신전을 준비합니다.
2. 플레이어를 팀에 배정합니다.
3. `/godwar start`를 실행합니다.
4. 플러그인이 인벤토리 초기화, 능력 배정, 팁 안내, 시작 설정 안내를 진행합니다.
5. 재추첨이 켜져 있으면 플레이어는 `/t yes` 또는 `/t no`로 능력을 확정합니다.
6. 확정 대기가 끝나면 시작 카운트다운이 진행됩니다.
7. 카운트다운 종료 후 게임룰과 월드 설정이 적용되고 전원이 팀 스폰으로 이동합니다.

`game.fast-start`가 켜져 있으면 준비 카운트다운은 `game.fast-ready-countdown-seconds`를 사용합니다. 꺼져 있으면 `game.ready-countdown-seconds`를 사용합니다.

## 능력 사용

대부분의 능력은 블레이즈 막대를 들고 사용합니다.

| 조작 | 의미 |
| --- | --- |
| 좌클릭 | 일반 능력 |
| 우클릭 | 고급 능력 |
| `/a` | 내 능력 GUI와 쿨타임 확인 |
| `/x <player>` | 타깃형 능력 대상 지정 |

`/godwar ability <player>`로는 본인 또는 같은 팀 플레이어의 능력만 볼 수 있습니다. 상대 팀 능력은 관리자만 확인할 수 있습니다.

능력 비용은 기본적으로 조약돌입니다. 스크루지 능력이 같은 팀에 있으면 팀원의 조약돌 비용이 절반으로 줄어듭니다.

## 심장 파괴 규칙

- 등록된 다이아몬드 심장만 팀 탈락 판정 대상입니다.
- 자기 팀 심장은 파괴할 수 없습니다.
- `core.require-empty-hand`가 켜져 있으면 심장은 맨손으로만 파괴할 수 있습니다.
- `core.pickaxe-unlock.*-seconds`가 `0` 이상이면 게임 시작 후 지정 시간이 지난 뒤 해당 곡괭이로 심장을 파괴할 수 있습니다.
- `core.forbid-diamond-pickaxe`가 켜져 있으면 다이아몬드 곡괭이로 심장을 파괴할 수 없습니다. 다만 다이아 곡괭이 시간 해제가 설정되어 있고 시간이 지났다면 허용됩니다.
- `core.protect-diamond-from-explosion`이 켜져 있으면 폭발로 심장이 제거되지 않습니다.

## 탈락과 중간 참여

팀이 탈락하면 `game.eliminated-player-action`에 따라 플레이어가 처리됩니다.

| 값 | 동작 |
| --- | --- |
| `spectator` | 관전 상태로 전환 |
| `kick` | 탈락 메시지와 함께 서버에서 퇴장 |
| `midjoin` | 살아있는 팀이 둘 이상이면 자동 중간 참여 시도 |

`game.allow-mid-join`이 켜져 있으면 관리자가 `/godwar midjoin <player> [team|auto]`로 진행 중인 게임에 플레이어를 중간 참여시킬 수 있습니다. 이미 살아있는 팀에서 정상 참여 중인 플레이어는 중간 참여 대상이 아닙니다.

## 운영 팁

- `/godwar status`로 상태, 팀킬, 우르프, 재추첨, 게임룰, 블랙리스트 개수를 확인합니다.
- `/godwar participants [검색어|팀]`로 참가자 팀/능력/킬 상태를 봅니다.
- `/godwar assignedabilities [검색어]`로 배정된 능력을 확인합니다.
- `/godwar skip [초]`로 능력 확정 대기를 강제로 종료하고 시작 카운트다운을 조정합니다.
- `/godwar gamerule apply`와 `/godwar gamerule restore`로 게임룰을 수동 적용/복구할 수 있습니다.
