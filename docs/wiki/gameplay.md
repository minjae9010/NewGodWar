# 게임 진행 가이드

NewGodWar의 기본 승리 조건은 자기 팀의 다이아몬드 심장을 지키면서 다른 팀의 심장을 파괴하는 것입니다. 심장이 파괴된 팀은 탈락하고, 마지막으로 남은 팀이 승리합니다.

## 게임 상태

| 상태 | 의미 |
| --- | --- |
| 대기 | 팀, 스폰, 심장 등 시작 설정을 준비하는 상태 |
| 준비중 | `/gw start` 이후 능력 배정과 재추첨 확정을 기다리는 상태 |
| 진행중 | 카운트다운이 끝나고 실제 전투가 진행되는 상태 |
| 종료 | 게임이 끝났거나 `/gw stop`으로 중단된 상태 |

## 팀과 신전

기본 팀은 `red`, `blue`, `green`입니다. `config.yml`의 `teams`에서 팀 표시 이름, 색상, 활성 여부를 조정할 수 있고, 설정 GUI에서도 주요 팀 설정을 관리할 수 있습니다.

각 활성 팀에는 두 가지 위치가 필요합니다.

| 위치 | 설정 방법 | 설명 |
| --- | --- | --- |
| 팀 스폰 | `/gw setspawn <team>` | 게임 시작과 리스폰 시 이동할 위치 |
| 다이아 심장 | `/gw settemple <team>` | 바라보는 다이아몬드 블록을 팀 심장으로 등록 |

활성 팀의 스폰이나 심장이 빠져 있으면 게임 시작이 거부됩니다.

## 시작 흐름

1. 관리자가 팀과 신전을 준비합니다.
2. 플레이어를 팀에 배정합니다.
3. `/gw start`를 실행합니다.
4. 플러그인이 인벤토리 초기화, 능력 배정, 팁 안내, 시작 설정 안내를 진행합니다.
5. 재추첨이 켜져 있으면 플레이어는 `/gw yes` 또는 `/gw no`로 능력을 확정합니다.
6. 확정 대기가 끝나면 시작 카운트다운이 진행됩니다.
7. 카운트다운 종료 후 게임룰과 월드 설정이 적용되고 전원이 팀 스폰으로 이동합니다.

`game.fast-start`가 켜져 있으면 준비 카운트다운은 `game.fast-ready-countdown-seconds`를 사용합니다. 꺼져 있으면 `game.ready-countdown-seconds`를 사용합니다.

## 능력 사용

대부분의 능력은 블레이즈 막대를 들고 사용합니다.

| 조작 | 의미 |
| --- | --- |
| 좌클릭 | 일반 능력 |
| 우클릭 | 고급 능력 |
| `/a` | 내 능력 GUI와 쿨타임 확인 |
| `/x <player>` | 타깃형 능력 대상 지정 |

`/gw ability <player>`로는 본인 또는 같은 팀 플레이어의 능력만 볼 수 있습니다. 상대 팀 능력은 관리자만 확인할 수 있습니다.

능력 비용은 기본적으로 조약돌입니다. 스크루지 능력이 같은 팀에 있으면 팀원의 조약돌 비용이 절반으로 줄어듭니다.

## 게임 시작 전 참고 규칙 및 팁

게임의 목표는 단순합니다. 우리 팀의 다이아몬드 심장을 지키면서 상대 팀의 심장을 먼저 파괴하면 됩니다. 심장이 부서진 팀은 탈락하고, 마지막까지 심장을 지킨 팀이 승리합니다.

시작 전에 아래 내용을 팀원들과 함께 확인해 두면 게임 흐름을 훨씬 빠르게 잡을 수 있습니다.

### 기본 규칙

- 각 플레이어는 빨강, 파랑, 초록 중 한 팀에 배정됩니다.
- 자기 팀 심장은 파괴할 수 없고, 상대 팀의 등록된 다이아몬드 심장만 탈락 판정 대상입니다.
- 기본 설정에서는 같은 팀을 공격할 수 없습니다.
- 능력은 무작위로 배정되며, 재추첨이 켜져 있으면 `/gw yes`로 확정하거나 `/gw no`로 다시 뽑을 수 있습니다.
- 대부분의 능력은 블레이즈 막대를 들고 좌클릭 또는 우클릭으로 사용합니다.
- 팀이 탈락하면 설정에 따라 관전 모드, 퇴장, 중간 참여, 그대로 둠 중 하나로 처리됩니다.

### 시작 전에 꼭 확인할 것

- `/a`로 내 능력 설명, 쿨타임, 사용 조건을 확인하세요.
- 타깃형 능력은 전투 전에 `/x <닉네임>`으로 대상을 미리 지정해 두면 좋습니다.
- 블레이즈 막대가 필요하면 막대기 3개를 세로, 가로, 또는 대각선으로 놓아 제작할 수 있습니다.
- 능력 비용은 기본적으로 조약돌이므로 초반에는 조약돌을 충분히 모아 두는 편이 좋습니다.
- 팀 채팅은 `/tc`로 켜고 끌 수 있습니다. 공격, 수비, 자원 담당을 팀 채팅으로 빠르게 정하세요.
- `/도박`을 사용할 수 있다면 조약돌로 추가 자원을 노릴 수 있지만, 초반 필수 자원까지 모두 쓰지는 않는 편이 안전합니다.

### 초반 운영 팁

- 시작 직후에는 한 명 이상이 심장 주변을 지키고, 나머지는 자원 확보와 정찰을 나누는 구성이 안정적입니다.
- 상대 심장 위치를 찾았더라도 혼자 무리해서 들어가기보다 팀원과 타이밍을 맞추는 편이 성공률이 높습니다.
- 능력 쿨타임이 길다면 먼저 상대를 빼내거나 수비를 흔든 뒤 중요한 순간에 사용하는 것이 좋습니다.
- 심장 파괴 조건은 서버 설정에 따라 달라질 수 있습니다. 맨손 파괴, 곡괭이 제한, 폭발 보호 여부를 시작 전에 확인하세요.
- 자신의 능력이 전투형이 아니어도 자원, 정찰, 수비, 타깃 지정으로 충분히 팀에 기여할 수 있습니다.
- 팀원이 탈락하거나 죽었을 때는 바로 위치와 상대 인원 정보를 공유하면 다음 교전에 도움이 됩니다.

## 심장 파괴 규칙

- 등록된 다이아몬드 심장만 팀 탈락 판정 대상입니다.
- 자기 팀 심장은 파괴할 수 없습니다.
- `core.require-empty-hand`가 켜져 있으면 심장은 맨손으로만 파괴할 수 있습니다.
- `core.pickaxe-unlock.*-seconds`가 `0` 이상이면 게임 시작 후 지정 시간이 지난 뒤 해당 곡괭이로 심장을 파괴할 수 있습니다.
- 금 곡괭이는 코어 파괴 허용 대상에서 제외됩니다.
- `core.forbid-diamond-pickaxe`가 켜져 있으면 다이아몬드 곡괭이로 심장을 파괴할 수 없습니다. 다만 다이아 곡괭이 시간 해제가 설정되어 있고 시간이 지났다면 허용됩니다.
- `core.protect-diamond-from-explosion`이 켜져 있으면 폭발로 심장이 보호됩니다. `core.explosion-unlock-seconds`가 `0` 이상이면 해당 시간이 지난 뒤 폭발로도 심장을 파괴할 수 있습니다.

## 탈락과 중간 참여

팀이 탈락하면 `game.eliminated-player-action`에 따라 플레이어가 처리됩니다.

| 값 | 동작 |
| --- | --- |
| `spectator` | 관전 상태로 전환 |
| `kick` | 탈락 메시지와 함께 서버에서 퇴장 |
| `midjoin` | 살아있는 팀이 둘 이상이면 자동 중간 참여 시도 |
| `none` | 플레이어의 팀, 위치, 게임모드를 그대로 유지 |

팀 탈락으로 자동 중간 참여가 발생하면 기존 인벤토리와 버프를 유지합니다. 배치 대상 팀은 살아있는 팀 중 인원이 가장 적은 팀을 우선하며, 인원이 같으면 해당 코어를 파괴한 플레이어의 팀을 우선합니다.

`game.allow-mid-join`이 켜져 있으면 관리자가 `/gw midjoin <player> [team|auto]`로 진행 중인 게임에 플레이어를 중간 참여시킬 수 있습니다. 이미 살아있는 팀에서 정상 참여 중인 플레이어는 중간 참여 대상이 아닙니다.

## 운영 팁

- `/gw status`로 상태, 팀킬, 우르프, 재추첨, 게임룰, 블랙리스트 개수를 확인합니다.
- `/gw participants [검색어|팀]`로 참가자 팀/능력/킬 상태를 봅니다.
- `/gw assignedabilities [검색어]`로 배정된 능력을 확인합니다.
- `/gw skip [초]`로 능력 확정 대기를 강제로 종료하고 시작 카운트다운을 조정합니다.
- `/gw gamerule apply`와 `/gw gamerule restore`로 게임룰을 수동 적용/복구할 수 있습니다.
